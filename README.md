# Scene and UI Interaction
## Project Description

This project demonstrates basic scene management and user interface interaction in Unity using a 2D (URP) setup. The project contains a main menu scene and a gameplay scene, connected through UI buttons.

## Features

* Two scenes: **MainMenu** and **GameScene**
* Start Game button to load the gameplay scene
* UI buttons to move the player left and right
* Return to Menu button to navigate back to the main menu
* Responsive UI using Canvas Scaler

## Project Structure

* **Scenes**

  * MainMenu
  * GameScene
* **Scripts**

  * SceneNavigation – handles scene transitions
  * PlayerMovement – controls player movement
* **GameObjects**

  * GameManager
  * Player (2D Square Sprite)
  * UI Buttons

## How to Run

1. Open the project in Unity.
2. Make sure MainMenu is set as the first scene in Build Settings.
3. Press Play.
4. Use the UI buttons to interact with the game.

## Learning Outcome

This project helps understand the basics of Unity scene creation, UI design, and connecting user interface elements to scripts for interaction.
